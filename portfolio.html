Notable Projects

Conjungo

By far my largest project, Conjungo is a social network for iOS. The back end, which features a handmade MySQL database and PHP enviornment, is hosted on Google App Engine. The initial version alone required tens of thousands of lines of code and took nearly a year to complete.

VRKit

VRKit is a system of classes for iOS that will allow developers to easily implement VR capabilities into their apps. I began this project very recently and many updates are on the way.

GraphingKit

I created this set of classes as a way of easily creating and displaying user accessible graphs in iOS applications. Any developer can use these classes to create full graphs with less than five lines of code.

Material Design for iOS

Inspired by Google's material design guidelines, I set out to replicate many UI elements for iOS. These classes allow developers to easily incorporate a number of iconic material design principles including layered views and interactive shadows.

Following Tech

Following Tech is a development blog I created where I post updates on my work and thoughts about future projects. This is also based on Google App Engine hosting, and relies on a MySQL and PHP back end that I developed.
